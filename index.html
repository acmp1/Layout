Basics

HTML - structure

  • Tags: container that defines the semantic of the content
  • Open-closure tags
  • Have attributes

CSS - style

  • #Id: target an unique element
  • .Class: target a group of elements

JS - changes

CSS Box Model

Margin: outside space of element

Border: around space of element

Padding: interior space of content


            /* CSS box model */
            .box{
              border: 25px solid white;
              padding: 25px;
              margin: 50px;
            }

Flex


            /* Flex */
            #flex-container{
              display: flex;
              background-color: #FF8519;
            }

            /* Screens < 800 px */
            @media all and (max-width: 800px) {
              #flex-container {
                /* By default row, changing to column */
                flex-direction: column;
              }
            }

Grid


            /* Grid */
            .grid-box{
              border: 25px solid white;
              padding: 25px;
            }

            #grid-box-center{
              background-color: #FF19CB;
              grid-column: 2;
              grid-row: 1 / 3;
            }

            #grid-container {
              display: grid;
              grid-template-columns: 300px 300px 300px;
              grid-template-rows: 400px 400px;
              grid-auto-flow: row;
              background-color: #CBFF19
            }
